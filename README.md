# FinpApp_console

Aplicación de consola para la gestión de finanzas personales.

## Funcionalidades

- Registrar categorías de gasto (FIJO / VARIABLE)
- Registrar movimientos (INGRESO / GASTO)
- Configurar presupuesto mensual
- Listar movimientos de un mes
- Ver resumen mensual:
  - Total de ingresos
  - Total de gastos
  - Saldo
  - Gastos por categoría

## Tecnologías

- Java 17+
- Repositorios en memoria
- Arquitectura modular (dominio, repositorio, servicio, UI)

## Ejecución

1. Abrir `FinanzasConsoleApp.java` en VS Code
2. Clic en **Run Java**
3. Usar el menú interactivo en consola
